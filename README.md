# BendMeter

**Spectrum waveform analyzer** by [bend music](https://bendmusic.com)

BendMeter displays a real-time scrolling waveform with 3-band frequency color mapping — Bass, Mid, and High — so you can read the energy of your mix at a glance. Resizable to any width, it fits anywhere in your session.

---

## Features

- Real-time scrolling waveform at 160fps
- 3-band frequency analysis — Bass (< 250 Hz) / Mid / High (> 4 kHz)
- 3 color schemes: **Bend** (yellow/orange/red) · **Vivid** (green/blue/pink) · **B&W**
- Dark and light mode
- Freely resizable — from a slim 350 × 100 bar to full-width 4096 × 800
- Zero latency — analyzer only, audio passes through unchanged
- Universal Binary — native Apple Silicon and Intel

---

## Installation

### macOS (VST3 / AU)

1. Download `BendMeter_macOS.pkg`
2. Double-click the installer and follow the steps
3. Restart your DAW
4. Scan for new plugins — BendMeter will appear under **bend music**

**Manual install:**
Copy `BendMeter.vst3` to `/Library/Audio/Plug-Ins/VST3/`
Copy `BendMeter.component` to `/Library/Audio/Components/`

### System Requirements

| | |
|---|---|
| macOS | 10.13 High Sierra or later |
| Architecture | Apple Silicon (M1+) and Intel (x86_64) |
| Formats | VST3, AU |
| DAW | Ableton Live 10+, Logic Pro X+, Reaper, Bitwig, and any VST3/AU host |

## Usage

| Control | Action |
|---|---|
| Color dots (top right) | Switch color scheme |
| Half-circle (middle right) | Toggle dark / light mode |
| **i** button (bottom right) | Show info |
| Drag corner | Resize window |
